## 📋 À propos du projet

**Notario Plus** est une plateforme SaaS conçue spécifiquement pour les notaires guinéens. Elle offre des outils numériques permettant de gérer efficacement les dossiers notariaux, d'automatiser la génération de documents officiels, et de centraliser le suivi des affaires juridiques au sein d'une interface moderne et sécurisée.

---

## ✨ Fonctionnalités principales

- 🗂️ **Gestion de dossiers notariaux** — Création, suivi et archivage des dossiers clients en toute simplicité
- 🔐 **Authentification des utilisateurs** — Connexion sécurisée avec gestion des rôles et des accès
- 📄 **Génération de documents** — Production automatisée de documents notariaux officiels
- 📊 **Tableau de bord** — Vue d'ensemble en temps réel de l'activité de l'étude notariale

## 🚀 Installation et démarrage

### Prérequis

- [Node.js](https://nodejs.org/) v18 ou supérieur
- [npm](https://www.npmjs.com/) ou [yarn](https://yarnpkg.com/)

### Étapes

```bash
# 1. Cloner le dépôt
git clone https://github.com/Notario-Guinee/notario-frontend.git

# 2. Accéder au dossier
cd notario-frontend

# 3. Installer les dépendances
npm install

# 4. Lancer le serveur de développement
npm run dev
```

L'application sera accessible à l'adresse : `http://localhost:3000`
